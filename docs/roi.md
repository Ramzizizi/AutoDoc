# Расчёт экономической эффективности (ROI)
## Проект «База правовой аналитики»

---

## 1. Описание эффекта

Система автоматизирует наиболее трудоёмкую часть работы юриста — поиск и систематизацию правовых материалов при подготовке заключения. Ключевые эффекты:

- **Сокращение времени поиска** за счёт полнотекстового поиска с морфологией, фасетной фильтрации и графа связей.
- **Автоматическая сборка .docx** — исключает ручное форматирование документа.
- **Накопление корпоративной базы знаний** — повторное использование ранее найденных материалов.

---

## 2. Исходные данные для расчёта

| Параметр | Значение | Обоснование |
|---|---|---|
| Среднее время подготовки заключения (до внедрения) | 5 ч | Опрос практикующих юристов |
| Среднее время подготовки заключения (после внедрения) | 1,5 ч | Экспертная оценка команды |
| Количество заключений в месяц (3 юриста) | 30 шт | Типовая нагрузка юридического отдела |
| Тарифная ставка юриста | 1 000 ₽/ч | Средняя по рынку РФ (2024) |
| Затраты на разработку | 150 000 ₽ | 15 нед × 5 чел × 40 ч × 500 ₽/ч |
| Затраты на сопровождение в год | 30 000 ₽ | Обновления, хостинг |

---

## 3. Расчёт

### 3.1. Экономия времени

```
ΔT = T_было − T_стало = 5 ч − 1,5 ч = 3,5 ч на заключение
```

### 3.2. Денежная экономия

```
E_мес = ΔT × количество_заключений × ставка
      = 3,5 × 30 × 1 000 = 105 000 ₽/мес

E_год = 105 000 × 12 = 1 260 000 ₽/год
```

### 3.3. Совокупные затраты (TCO за год)

```
TCO_год = C_разр + C_сопр = 150 000 + 30 000 = 180 000 ₽
```

### 3.4. Срок окупаемости

```
T_окуп = TCO_год / E_мес = 180 000 / 105 000 ≈ 1,7 месяца
```

### 3.5. ROI за первый год

```
ROI = (E_год − TCO_год) / TCO_год × 100%
    = (1 260 000 − 180 000) / 180 000 × 100%
    = 600%
```

## 5. Дополнительные нефинансовые эффекты

| Эффект | Описание |
|---|---|
| Снижение правового риска | Меньше вероятность пропустить применимую норму или релевантный прецедент |
| Сохранение знаний | Корпоративная база накапливается и не теряется при увольнении сотрудника |
| Унификация | Единый формат заключений vs. разнородные Word-файлы у каждого юриста |
| Быстрый онбординг | Новый сотрудник сразу использует накопленную базу материалов |
| Масштабируемость | При росте нагрузки систему проще масштабировать, чем нанимать новых сотрудников |

---

## 6. Формула ROI (из методички)

По формуле методического указания:

```
ROI = (Результат − Затраты) / Затраты × 100%
```

Где:
- **Результат** = суммарная экономия трудозатрат в денежном выражении за расчётный период
- **Затраты** = совокупная стоимость владения (TCO) за тот же период
